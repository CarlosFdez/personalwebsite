Personal Programming Projects

Sometimes I'll contribute to someone else's existing project or create something new. These are a few.

  • VideoCrop

    When I record videos, I have a large number of footage I need to sift through before editing. I created a video preview and cropping tool that that can cut without a retranscode. It allows me to reduce my footage and hard disk usage before I begin video editing. This was my first attempt at a native desktop application. Written in C++ and Qt.

  • SpueBox

    A custom bot for the discord chat client. It's used primarily to play audio clips from various online sources (including Youtube) in voice chat rooms. Written in Python using the discord.py library.

  • MonsterHunter4UDatabase

    Contribution: Added a universal search to an existing Android application written in Java that provides data about the game Monster Hunter 4 Ultimate. Also contributed a few other bug fixes and tweaks.

  • PokeBattle Sim

    Contribution: Helped with a Pokemon battle system simulator for those that wanted quick online matches without the time burden of creating teams using the actual game. The simulator was written in CoffeeScript, ran on NodeJS, and used Backbone for the front end. Failed after those involved lost interest due to growing responsibilities and competition from Pokemon Showdown.

About this site

This is a regular static website built using html, css, and vanilla javascript. Written using the Atom text editor and tested using the live-server node package. It is hosted using nginx on an Ubuntu instance in Digital Ocean. No frameworks were harmed in the making of this website.
