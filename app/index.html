Programming Projects

I've created a few applications in my free times for personal reasons. These are some of my favorites.

  • VideoCrop

    When I record videos, I have a large number of footage I need to sift through before editing. I created a video preview and cropping tool that that can cut without a retranscode. It allows me to reduce my footage and hard disk usage before I begin video editing. This was my first attempt at a native desktop application. Written in C++ and Qt.

  • SpueBox

    A custom bot for the discord chat client. It's used primarily to play audio clips from various online sources (including Youtube) in voice chat rooms. Written in Python using the discord.py library.

  • MonsterHunter4UDatabase

    Contribution: Added a universal search to an existed Android application written in Java that provides data about the game Monster Hunter 4 Ultimate. Also contributed a few other bug fixes and tweaks.

  • PokeBattle Sim

    Contribution: Helped with a failed Pokemon Battle Simulator for those that wanted quick online matches without the time burden of creating teams using the actual game. The simulator was written in CoffeeScript running on NodeJS, and used Backbone for the front end. Project owners lost interest especially with Pokemon Showdown maintaining a large user share.
