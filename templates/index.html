Personal Programming Projects

These are a few of the projects I've created.

  • VideoCrop

    A graphical tool I created to assist with my video editing hobby. Used to preview footage and cut on keyframes to avoid losing quality due to generational loss. This allows me to reduce the footage I need to examine during the editing phase, and reduce my hard disk usage as a bonus. Written in C++ and Qt.

  • SpueBox

    A custom bot for the discord chat client. It's used primarily to play audio clips from various online sources (including Youtube) in voice chat rooms. Written in Python using the discord.py library.

  • tmibot.js

    A node library used to create chatbots for Twitch. Wraps the existing tmi.js library to add command parsing.

About this site

This is a regular static website created using html, css, and vanilla javascript. I built the website using the Atom text editor, and tested it using the live-server node package. It is hosted using nginx on an Ubuntu instance in Digital Ocean.

This website's javascript was written from scratch to demonstrate that I know how to work with the DOM. However, if the scope of this portfolio grows I may end up using one in the future. If I do, I'll use either JQuery or React.

The social network icons belong to Font Awesome. No frameworks were harmed in the making of this website.
